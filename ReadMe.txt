This is a case study project on which investors can fund on a particular start up 
considering different scenarios we have tried to find out the information 
analyzed the dataset and visualized it using python libraries


Python libraries used  - > 

- csv - to load csv file dataset
- numpy and pandas - to fetch the required information
- matplotlib - to visualize the fetched information

PreRequirements to run this project: - 

- Anconda should be installed in the PC as it has all the required libraries for this project pre installed
- also Anaconda has certain softwares one of which is jupyter notebook 

open the code python notebook file using jupyternotebook 
keep the startup_funding.csv in the same folder of the notebook file or change the path
